About This Content

Sierra is a rule-based static analysis toolkit supporting team collaboration. It integrates widely used open source tools, such as FindBugs and PMD, into a single team-oriented tool.

SureLogic's Sierra offering is Eclipse Ready™ (www.eclipse.org). This offering can be installed and used with other Eclipse 3.2- and Eclipse 3.3-based offerings.
